# Radius

8 stops, t-shirt sized. Names mirror Figma's `radius/*` variables — `var(--radius-md)` is 8px both places. T-shirt sizing here (rather than pixel-numbered like spacing/typography) because radius decisions are role-driven: you reach for "sm" or "md" by intent, not by exact pixel value.

<div style={{ display: 'grid', gridTemplateColumns: 'repeat(4, 1fr)', gap: '24px', marginTop: '24px' }}>
  {[
    { name: 'radius-none', value: '0',       displayPx: 0  },
    { name: 'radius-xs',   value: '2px',     displayPx: 2  },
    { name: 'radius-sm',   value: '4px',     displayPx: 4  },
    { name: 'radius-md',   value: '8px',     displayPx: 8  },
    { name: 'radius-lg',   value: '12px',    displayPx: 12 },
    { name: 'radius-xl',   value: '16px',    displayPx: 16 },
    { name: 'radius-2xl',  value: '24px',    displayPx: 24 },
    { name: 'radius-full', value: '9999px',  displayPx: 40 },
  ].map(({ name, value, displayPx }) => (
    <div key={name} style={{ display: 'flex', flexDirection: 'column', gap: '8px' }}>
      <div style={{
        width: '80px',
        height: '80px',
        backgroundColor: '#1a00ff',
        borderRadius: `${displayPx}px`,
      }} />
      <div style={{ fontFamily: 'JetBrains Mono, monospace', fontSize: '12px' }}>
        <strong style={{ color: '#181a1e' }}>{name}</strong>
      </div>
      <div style={{ fontFamily: 'JetBrains Mono, monospace', fontSize: '12px', color: '#64686d' }}>
        {value}
      </div>
    </div>
  ))}
</div>

## Semantic roles

| Role | Token | Pixels | Use |
|---|---|---|---|
| Card | `--radius-card` | 4 | Cards, panels |
| Control | `--radius-control` | 8 | Inputs, buttons (default) |
| Pill | `--radius-pill` | full | Pills, avatars, capsule buttons |
